
## Overview
The artifact provides code and scripts to reproduce representative experimental
results reported in the paper. It includes the main processing pipeline and
evaluation components used in our experiments.

To preserve double-blind review, the artifact is provided in an anonymized and
self-contained form.

## Environment
The artifact has been tested under the following environment:
- OS: Ubuntu 20.04
- Python: 3.9+

No specialized hardware is required.
